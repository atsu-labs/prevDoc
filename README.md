# 建築図面注釈・計測ソフトウェア 取扱説明書

本ソフトウェアは、既存のPDF図面を背景として読み込み、その上に距離計測、面積計算、注釈（テキスト・図形）の追加を行うためのデスクトップアプリケーションです。

## 1. 主な機能

### A. ファイル操作
- **PDFインポート:** 図面PDFを背景として読み込みます（複数ページ対応）。
- **背景差し替え:** 描画した注釈データを維持したまま、背景のPDFファイルのみを更新します。
- **プロジェクト保存/読込:** 編集中のデータ（注釈、スケール設定）をJSON形式で保存し、後から再編集できます。
- **PDF書き出し:** 背景図面と注釈を合成した新しいPDFを出力します。

### B. 計測・描画ツール
- **キャリブレーション:** 図面上の既知の2点間を指定し、実寸法を入力することで縮尺（1pxあたりのmm数）を定義します。
- **距離計測:** 直線距離を計測し、実寸法をラベルとして表示します。
- **面積計測:** 多角形で囲んだ領域の面積（m²）を自動計算します。
- **15m円描画:** クリックした地点を中心に、半径15m（実寸）の円を描画します。
- **テキスト注釈:** 任意の場所にテキストを追加します。

### C. オブジェクト編集
- **選択・移動:** 配置済みのオブジェクトを選択し、ドラッグで位置を微調整できます。
- **プロパティパネル:** 画面右側のパネルで以下の属性をリアルタイムに編集できます。
  - テキスト内容
  - フォントの種類（システムフォントから選択）
  - 文字の大きさ（図面に固定してスケーリングされます）
  - 表示色
- **削除:** 不要になったオブジェクトをパネル上のボタンから削除できます。

## 2. セットアップと実行方法

### 必要環境
- Python 3.10以上
- 依存ライブラリ: `PySide6`, `PyMuPDF` (fitz), `qtawesome`

### インストール
```powershell
pip install PySide6 pymupdf qtawesome
```

### 実行
```powershell
python src/main.py
```

---

## 3. 基本的な操作フロー

1.  **図面を開く:** ツールバーの「PDFを開く」から図面を選択します。
2.  **縮尺を設定する:** 「キャリブレーション」ボタンを押し、図面のスケールバーなどの既知の2点をクリックします。ポップアップに実寸法（mm）を入力します。
3.  **計測・描画を行う:** 任意のツール（距離、面積、15m円など）を選択して図面上に書き込みます。
    - *面積計測は右クリックで形を確定します。*
4.  **微調整する:** 「選択・移動」モードに切り替え、オブジェクトを選択して右側のパネルで文字サイズや色を調整したり、位置を動かしたりします。
5.  **結果を出力する:** 「保存」で作業データを残すか、「PDF書き出し」で成果物を出力します。

## 4. 技術仕様（開発者向け）

### アーキテクチャ
- **GUI:** PySide6 (Qt for Python)
  - `QGraphicsView` / `QGraphicsScene` を使用した高精度な座標管理。
- **PDF処理:** PyMuPDF (fitz)
  - 150 DPIでの高解像度レンダリング。
- **データ構造:** 
  - 座標系はPDFのピクセル座標に基づき、計算時に `scale_factor` を適用。
  - テキストサイズは「図面固定」モード（拡大縮小に連動）。

### ファイル構成
- `src/main.py`: 起動用のエントリーポイント。
- `src/firepreview/main_window.py`: メインウィンドウと画面全体の制御。
- `src/firepreview/ui/`: キャンバスやサイドパネルなどのUIコンポーネント。
- `src/firepreview/services/`: PDF入出力やプロジェクト保存などのアプリケーションサービス。
- `src/firepreview/models.py`: データモデル（Annotation）、シリアライズ処理。
